validaPotencialMapa :: [(Peca, Coordenadas)] -> Bool #

Função principal que vai verificar se uma lista de peças e respetivas coordenadas podem construir um mapa válido.

Exemplo:

>>> validaPotencialMapa [(Porta,(0,3)),(Bloco,(0,4)),(Bloco,(1,4)),(Bloco,(2,4)),(Bloco,(3,4)),(Bloco,(4,4)),(Caixa,(4,3)),(Bloco,(5,4)),(Bloco,(6,4)),(Bloco,(6,3)),(Bloco,(6,2)),(Bloco,(6,1))]
True
>>> validaPotencialMapa []
False
